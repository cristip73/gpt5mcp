# GPT-5 MCP Server

**gpt5-server** - OpenAI GPT-5 API integration for text generation
- Environment: `OPENAI_API_KEY`
- Command: `node build/index.js`
- Tools: `gpt5_generate`, `gpt5_messages`

## Environment Variables
API key is stored in `.env`:
- `OPENAI_API_KEY`
